# Pre Requirements
- [Django](https://www.djangoproject.com/)
- [Node.js](https://nodejs.org/en/)

# Install

```terminal
npm install
```

```terminal
npm run build
```

```terminal
python3 manage.py migrate
```

```terminal
python3 manage.py runserver
```
